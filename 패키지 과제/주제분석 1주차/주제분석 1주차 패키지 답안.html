0. 들어가기 전에¶

0.1 데이터 소개¶

주어진 데이터 파일은 '직방'에서 제공한 데이터로, Dacon 4회 컴피티션 '아파트 실거래가 예측'의 데이터입니다. 이 데이터에는 우리가 예측하고자 하는 'Y : 아파트실거래가'와 아파트 거래인, 지역, 전용면적, 공급면적 등의 정보가 포함됩니다. 대회 직후 연습용으로 바뀐 데이터 셋이니 사용해도 문제없습니다!

1,000,000 여 개의 train data와 500,000 여 개의 test data가 주어집니다. 하지만 컴피티션 특성상 대회 후 연습용이더라도 test data에는 Y가 포함되어 있지 않습니다. 따라서 train data를 일부 분리해서 평가하려 합니다. 아파트 거래일 기준으로 2008년부터 2017년까지 데이터가 존재하기 때문에, 상대적으로 가벼운 환경에서 분석을 위해 데이터 셋을 일부 제한해서 모델을 만들 계획입니다.

0.2 1주차 패키지 목표¶

클린업 패키지는 R을 기반으로 만들었지만, 주제분석 패키지는 Python을 기반으로 만들었습니다. 따라서 기본적으로 파이썬에서 데이터프레임을 다루는 Pandas를 익숙하게 하고, matplotlib을 통한 시각화를 다루려 합니다.

데이터 탐색과 시각화 후, 여기서 얻은 인사이트를 바탕으로 전처리를 시행하는 과정들을 거칠 예정입니다. 물론 우리가 데이터 분석을 하면 예상되는 결과와 실제 결과가 다를 경우도 많고, 이번 패키지도 그렇습니다... 하지만 분석의 흐름과 논리를 조금은 더 익숙하게 만들어 드리고 싶네요...!

더불어서 이미 클린업을 통해 여러 이론적인 내용들을 다뤘기 때문에, 조금 다양한 방식의 전처리들을 다루려 합니다. 시계열 주기성을 고려한 인코딩, 텍스트 데이터 전처리, 범주형 자료 인코딩 등이 포함될 예정입니다. 어려울 수 있지만 최대한 힌트와 함께 구성될 예정이니 따라와 주세요!

2. 데이터 시각화¶

해야하니까 하는 EDA를 넘어서, 파생변수를 생성하는데에 있어서 유의미한 인사이트를 얻기 위해 노력해봅시다.

일단 그려보기도 하지만, 분석과 관계된 가설을 파악하기 위해 시각화를 진행하는 경우가 많습니다.

  • 만약 matplotlib이 너무 불편하면 python의 ggplot2인 plotline을 사용해도 괜찮습니다.
    • 다만 어렵지 않은 시각화 위주로 가려하니, 파이썬의 방식에 익숙해져봅시다.
  • matplotlib과 더불어 seaborn을 이용해도 괜찮습니다. 색과 같이 세부적인 것은 똑같지 않아도 됩니다.
  • 오늘의 시각화는 '예쁘게 하기'보다 '일단 그리기'에 초점을 맞췄습니다.
    • 예시에서는 matplotlib과 seaborn을 왔다갔다 하니까, 큰틀에서만 똑같게 그려주시면 됩니다.
  • 추가적으로 예쁘게 그리시면 우수 답안으로 선정될 가능성이 높아집니다.
  • 모든 플랏에는 간단하게나마 해석을 덧붙여 주세요!

3. 파생변수 생성¶

3.1 아파트 연차 변수 생성¶
